### Directoy commands
- ```pwd``` --> print work directory. Gives the present working directory.

![Screenshot from 2024-10-01 18-59-36](https://github.com/user-attachments/assets/d5bf2c42-4089-4bd3-b6f3-9e556787a9c3)


- ```cd path_to_directory``` --> change directory to the provided path

![Screenshot from 2024-10-01 18-58-33](https://github.com/user-attachments/assets/2fa28ee9-3f7b-43a9-a0eb-3dc5d13ce9d7)


- ```cd ~ ``` or just  ```cd ``` --> change directory to the home directory

![Screenshot from 2024-10-01 18-59-54](https://github.com/user-attachments/assets/dfee72ea-7d8c-4663-a88c-d9f2b9744810)

![Screenshot from 2024-10-01 19-00-28](https://github.com/user-attachments/assets/bcde7fb8-3407-40ff-af08-9a8b0f20d8bd)

![Screenshot from 2024-10-01 19-00-45](https://github.com/user-attachments/assets/a62d7d90-650a-491f-b496-d6151c18fc60)

- ``` cd ../..``` --> Change directory to 2 levels back.
